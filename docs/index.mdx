**webtensor** is a multi-backend tensor library built for the browser. Train, visualize, and run inference with WebGPU acceleration as a TypeScript library. Backends: CPU (TypeScript), WASM (Rust via wasm-pack), and WebGPU (WGSL compute shaders). All three share full kernel parity across 16 ops.

> **Status: active development — building the foundation.**

## Architecture at a glance

```
user code  →  @webtensor/core  (Tensor, ops, autograd, compileGraph)
                    ↓
           @webtensor/runtime  (Engine, Backend interface, view ops)
                    ↓
   backend-cpu | backend-wasm | backend-webgpu
```

The same `Graph` IR runs on any backend. Backends register themselves at import time; `Engine.create('cpu' | 'wasm' | 'webgpu')` picks one.

See [Architecture](/reference/architecture) for the full contract and [IR reference](/reference/ir-reference) for the graph schema.
